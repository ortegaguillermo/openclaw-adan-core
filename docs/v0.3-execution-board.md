# v0.3 Execution Board — skill-synthesizer MVP

## Objective
Ship an optional, feature-flagged `skill-synthesizer` module that generates safe, reviewable skill scaffolds.

## Scope

### In Scope (MVP)
- Structured request intake (`skill-request.yaml`)
- Deterministic scaffold generation
- Valid `SKILL.md` frontmatter + trigger description
- Human review checklist before adoption

### Out of Scope (MVP)
- Auto-execution of generated scripts
- Auto-publish / auto-merge
- Autonomous external actions

## Work Items

1. **Module scaffold**
   - Status: ✅
   - Deliverable: `modules/skill-synthesizer/*`

2. **Template contract**
   - Status: ✅
   - Deliverable: `skill-request.example.yaml`, `SKILL.template.md`

3. **Review gate**
   - Status: ✅
   - Deliverable: `REVIEW-CHECKLIST.md`

4. **Feature-flag integration docs**
   - Status: ✅
   - Deliverable: README + FEATURES updates

5. **Adoption walkthrough**
   - Status: ✅
   - Deliverable: module README with copy/apply flow

## Acceptance Criteria
- A user can fill `skill-request.example.yaml`
- A user can produce a draft skill folder from templates
- Resulting `SKILL.md` follows required name/description frontmatter
- No step requires platform-specific commands
- No autonomous risky action is enabled by default

## Rollout
- Keep `meta_skill_synthesizer=false` by default
- Test in report-only mode first
- Promote to team usage after 2 successful manual reviews
